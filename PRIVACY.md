# Privacy Policy for Fable Xport Extension

Effective date: April 3, 2026

Fable Xport Extension ("the extension") helps users export their signed-in Fable library data to CSV files for Goodreads workflows.

## Data We Process

The extension may process the following data when you use export features:

- Authentication/session data from Fable (for example, auth token/session context and account identifier) to access Fable APIs.
- Library data from your Fable account, such as lists, books, ratings, and review-related fields needed to build the CSV export.
- Export preferences you choose in the popup (for example, selected list IDs and export format).

## How We Use Data

Data is used only to provide extension functionality, including:

- Authenticating with Fable using your existing signed-in browser session.
- Fetching account/list/book/review data from Fable endpoints required for export.
- Generating downloadable CSV files.

## Storage

The extension may store limited session/context data locally in your browser via Chrome extension storage to make repeated exports smoother.

The extension does not operate its own remote database for storing your exported library data.

## Data Sharing and Selling

- We do not sell your data.
- We do not use your data for advertising.
- We do not use your data for creditworthiness or lending purposes.
- We do not share your data with third-party analytics or ad platforms.

## Network Access

The extension sends requests only to Fable domains needed to perform exports, including:

- https://fable.co
- https://api.fable.co

## Security and Code Execution

- The extension does not execute remote code.
- The extension runs packaged local code and requests data only as required for export functionality.

## StoryGraph Compatibility Note

The extension provides Goodreads-focused CSV export.  
For StoryGraph import compatibility, users should import to Goodreads first and then import the Goodreads export into StoryGraph.
